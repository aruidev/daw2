## Important

#### Estils

- `input.css` és el css base.  
Desde allà definim les clases base, importem Tailwind i el plugin de DaisyUI (utilitzat únicament per la funcionalitat del accordion a la secció de FAQ).
- `output.css` són els estils generats per tailwind i per tant no són llegibles ni editables manualment.

#### HTML

Per poder treballar amb components reutilitzables, s'ha utilitzat la tècnica d'importar components JavaScript amb string templates.
